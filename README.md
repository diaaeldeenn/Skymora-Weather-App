## ğŸ§  How It Works  

- User enters a city name in the search bar.  
- A request is sent to **WeatherAPI** (3-day forecast).  
- JavaScript extracts data including:  
  - Temperature  
  - Condition text  
  - Humidity  
  - Wind speed  
  - Weather icons  
- Weather cards are dynamically generated and inserted into the DOM.  
- Custom CSS effects create the neon holographic appearance with glow, gradients, and animations.
